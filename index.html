Article category

Cover story block

This is the cover story block and can be used by tagging an article with Cover story on the WordPress site. There should only be one of these on the page, and this must be the first article in the email unless using the sponsored post block instead. The image bleeds edge-to-edge which make it stand out among the rest, including on mobile. Do not use this when you mean to use the featured post block.
 
 
 

Article category

Featured post block

This is a featured post block and can be used by tagging an article with Featured on the WordPress site. Use it to call out an article and give it more prominence, or to add some variety. Do not use it in place of the cover story block.

Sponsored post block

This block is for the main story in a sponsored report, or for a round table discussion. Use this block by tagging an article with Sponsored on the WordPress site. It has space for a logo which can be edited to the desired size.

Basic post block

The default block that posts will be added to. Use this block to add a missing/broken post or to make a featured post into a smaller one if needed. Remember to link the title and the image to the article.
